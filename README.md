LoanApp Hariz is a simple Android application designed to help users calculate car loan details quickly and accurately. The app allows users to enter the vehicle price, down payment, loan duration, and interest rate, then automatically computes the loan amount, total interest, total payment, and monthly installment. Built using Java and XML in Android Studio, the project focuses on clean UI design, basic financial calculation logic, and beginner-friendly app structure. It also includes a toolbar menu and an About page that displays project information and a GitHub link.
